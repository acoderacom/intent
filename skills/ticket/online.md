# Online Ticket Format (Linear/GitHub/Jira/Asana)

Title: `[INT-YYYYMMDD-HHMMSS] {intent summary}`

Body:
```markdown
**Status:** Backlog

**Intent:** [what user wants]

**Context:** [relevant files, patterns]

**Constraints:** Use: [list] | Avoid: [list]

**Assumptions:** [AI's guesses - surfaces misalignment]

**Tasks**
- [ ] Task 1 description
- [ ] Task 2 description

**Definition of Done**
- [ ] Criterion 1
- [ ] Criterion 2

**Change Class:** A|B|C - [reason]
```

## Status Updates

Use adapter's Update tool (from Intent Config) to change status and update task checkboxes.
